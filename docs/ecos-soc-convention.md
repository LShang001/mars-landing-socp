# ECOS SOC 锥约束符号约定

> 适用于所有使用 ECOS 求解器的 SOCP 项目（火星着陆 TrajPlan、VTVL MPC 等）。
> 符号写反是最常见的隐蔽 bug — 矩阵维度/非零元数全对，但物理意义全错。

---

## 一、ECOS 的锥约束接口

ECOS 对不等式约束矩阵 G 的解释分两段：

| 行范围 | 类型 | 解释 |
|--------|------|------|
| `0 ~ l-1` | 线性约束 | `G[i,:] @ x ≤ h[i]` |
| `l ~ m-1` | 锥约束 | `h[i:i+q[j]] - G[i:i+q[j],:] @ x ∈ K_q[j]` |

其中 `q[j]` 是第 j 个锥的维度，`K_q` 是 q 维二阶锥：
```
K_q = { (x₀, x₁, ..., x_{q-1}) : x₀ ≥ sqrt(x₁² + ... + x_{q-1}²) }
```

**关键点**：锥约束是 `h - Gx ∈ K`，不是 `Gx ≤ h`。G 的符号决定了 `h - Gx` 的符号。

---

## 二、常见锥约束的正确写法

### 2.1 下滑角锥：`||[ry, rz]||₂ ≤ rx·tan(θ)`

需要 `h - Gx = [rx·tanθ, ry, rz] ∈ K₃`。

**正确写法**（G 用负系数）：
```
G 行 1: [-tanθ, 0, 0, ...]  h[1] = 0
G 行 2: [0, -1, 0, ...]     h[2] = 0
G 行 3: [0, 0, -1, ...]     h[3] = 0
```
→ `h - Gx = [rx·tanθ, ry, rz]` ✅

**错误写法**（G 用正系数，`mars_model.py` 旧版曾犯）：
```
G 行 1: [tanθ, 0, 0, ...]   h[1] = 0
G 行 2: [0, 1, 0, ...]      h[2] = 0
```
→ `h - Gx = [-rx·tanθ, -ry, -rz]` ❌
→ 锥条件变为：`-rx·tanθ ≥ sqrt(ry²+rz²)` → 要求 rx ≤ 0（违反物理！）

### 2.2 推力松弛锥：`||[ux, uy, uz]||₂ ≤ σ`

需要 `h - Gx = [σ, ux, uy, uz] ∈ K₄`。

**正确写法**（G 用负系数）：
```
G 每行分别 = [-1, 0, 0, 0, ...] 对应 σ
            [0, -1, 0, 0, ...] 对应 ux
            [0, 0, -1, 0, ...] 对应 uy
            [0, 0, 0, -1, ...] 对应 uz
h 全为 0
```
→ `h - Gx = [σ, ux, uy, uz]` ✅

### 2.3 质量值不等式（线性约束）

线性约束是 `Gx ≤ h`，不需要 SOC。例如：
```
G: [-μ₁, -1] 对应 [z, σ]
h: -μ₁(1+z₀)
```
→ `-μ₁·z - σ ≤ -μ₁(1+z₀)` → `μ₁(z - z₀ - 1) + σ ≥ 0` ✅

## 三、CasADi 到 ECOS 的矩阵提取

### 3.1 约定

定义 CasADi 符号约束向量 `eq` 和 `ineq`：
- 等式约束：`eq(x) = 0`（如 `rx(0) - r0 = 0`）
- 不等式约束：`ineq(x) ≤ 0`（如 `-μ₁(z - z₀ - 1) - σ ≤ 0`）

### 3.2 提取公式

```python
A  = jacobian(eq, x)(x=0)    # 稀疏 Jacobian — 自动处理
G  = jacobian(ineq, x)(x=0)
b  = -eq(0)                   # ⚠️ 注意负号！不是 -A@x+eq
h  = -ineq(0)                 # ⚠️ 注意负号！
```

**为什么是 `-eq(0)` 而不是 `-A@x + eq(0)`？**

两者在 x=0 处等价（因为 A@0 = 0），但 `-eq(0)` 更简洁且不容易出错。旧版 `mars_model.py` 曾用 `-A@x+eq`，在引入符号变量后能正常运行，但逻辑上多了一步不必要的矩阵乘法。

### 3.3 验证检查清单

生成矩阵后，验证以下关键值：

| 检查点 | 预期值 | 含义 |
|--------|--------|------|
| `b[0]` | `r₀[0] = 1500` | 初始位置 |
| `b[6]` | `ln(m₀) = 7.552` | 初始质量对数 |
| `b[13]` | `½g·dt² = 13.528` | rx 动力学常数项 |
| `h[0]` | `-μ₁(1+z₀) < 0` | 质量下界 |
| `h[124]` | `0` | 下滑角锥起点 |
| `h[217]` | `0` | 推力锥起点 |

`h` 向量的 SOC 锥段（行 124-340）必须全为 0。如果不是，检查约束排序或符号。

## 四、跨项目适用性

本约定适用于所有使用 ECOS 的 GNC 项目：

| 项目 | 问题类型 | 涉及的锥 |
|------|----------|----------|
| `ecos-cn-raspberry/MarsLanding` | 火星着陆 SOCP | 下滑角锥(q=3), 推力锥(q=4) |
| `EVTVL_TPMPC_raspberrypi/TrajPlan` | VTVL 轨迹规划 SOCP | 推力锥(q=4) |
| `EVTVL_TPMPC_LattePanda/TrajPlan` | VTVL 轨迹规划 SOCP | 推力锥(q=4) |

三个项目共享相同的 `h-Gx ∈ K` 约定和 CasADi 提取公式。
